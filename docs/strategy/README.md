## 策略通用说明

### 数据来源
- 行情数据：Futu API (富途牛牛)
- 历史数据：掘金量化 (GM) 数据

### 标的代码格式
- Futu 格式：`SH.600000`, `SZ.000001`
- GM 格式：`SHSE.600000`, `SZSE.000001`
